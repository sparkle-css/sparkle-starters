Thank you for using Sparkle !

Currently on version: Latest

Responsive

Sparkle is a responsive framework that uses the mobile-first approach, which ensures proper mobile view compatibility.

Modular

Sparkle offers you the possibility to only import the elements you need. Simply comment out what you do not wish to render, then run the provided gulp or npm run build command!

User-Friendly

Sparkle strives to remain as user-friendly as possible, using semantics that can be easily remembered and used by anyone.

Open-Source

Most importantly, Sparkle is free to use! No fees, ever! And because it is open-source, you are free to request more features and to contribute to the improvement of the framework!
